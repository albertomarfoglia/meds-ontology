Codec back to ToC or Class ToC

IRI: https://albertomarfoglia.github.io/meds-ontology#Code

Metadata/vocabulary entry describing a measurement code (code string, description, parent codes, external links).

is in domain of

hasCode op, parentCode op

LabelSamplec back to ToC or Class ToC

IRI: https://albertomarfoglia.github.io/meds-ontology#LabelSample

A prediction sample row: subject, prediction_time, and one label value (boolean/integer/float/categorical).

Measurementc back to ToC or Class ToC

IRI: https://albertomarfoglia.github.io/meds-ontology#Measurement

A single observation/row in MEDS: (subject, time?, code, value modalities...). Time is nullable for static measurements.
has super-classes
is in domain of
code String dp, describedInDataset op, hasCode op, hasSubject op, hasValueModality op, numericValue dp, siteId dp, textValue dp, time dp
is disjoint with
LabelSample c

Subjectc back to ToC or Class ToC

IRI: https://albertomarfoglia.github.io/meds-ontology#Subject

Primary entity for which care observations are recorded. Maps to MEDS subject_id (patient or admission-level subject).
is in domain of
assignedSplit op, describedInDataset op, subject Id dp
is in range of
hasSubject op
has keys
subject Id dp

SubjectSplitc back to ToC or Class ToC

IRI: https://albertomarfoglia.github.io/meds-ontology#SubjectSplit

is in range of

has members
held_out ni, train ni, tuning ni
